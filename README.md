# Netflix Data Visualization using Python

This project analyzes and visualizes the Netflix Movies and TV Shows dataset using Python.

## Project Overview

The goal of this project is to explore Netflix content data and generate meaningful visualizations using Pandas and Matplotlib.

The dataset contains information about Netflix movies and TV shows including:

- Title
- Type (Movie / TV Show)
- Release Year
- Rating
- Duration
- Country

## Tools & Libraries

Python  
Pandas  
Matplotlib  
Kaggle Dataset  

## Data Source

Netflix Titles Dataset from Kaggle.

## Visualizations

This project includes several visualizations such as:

1. Movies vs TV Shows distribution
2. Content rating percentage
3. Movie duration distribution
4. Release year vs number of shows
5. Top 10 countries by Netflix content
6. Comparison of Movies and TV Shows released over years
